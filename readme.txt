Use shortcodes and template tags to get data from Sphere FAF in WordPress.

== Description ==

Integrates Blackbaud Sphere Friends Asking Friends with WordPress by pulling participant reports and returning the values in WordPress for custom leaderboards using both template tags and shortcodes.

== Installation ==

1. Unzip and upload the `blackbaud-sphere` directory to `/wp-content/plugins/`
2. Activate the plugin through the `Plugins` menu in WordPress
3. There is no step 3.\*
